# Customer Segmentation Study

This project focuses on conducting **Customer Segmentation Analysis** for an online retail business using data analysis methods. The main aim is to classify customers into different groups based on their purchasing patterns and demographic details to support focused marketing strategies and better business decisions.

## Dataset

* ifood_df.csv

The dataset includes customer demographic details, purchase records, and behavioral information necessary for performing segmentation analysis.

## Objectives

* Conduct data exploration and data cleaning
* Examine customer purchasing patterns using statistical summaries
* Implement clustering methods to categorize customers
* Visualize the different customer groups through charts and graphs
* Extract insights and suggest practical business recommendations

## Methodology

* Data preparation and preprocessing
* Selection of relevant features for clustering
* Data normalization using StandardScaler
* Customer grouping using K-Means clustering algorithm
* Visualization of the resulting customer clusters
* Analysis and interpretation of cluster characteristics

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

## Results

The customers were grouped into multiple segments based on factors such as income level, spending habits, and purchasing behavior. Each segment reflects a different customer type that can be targeted with tailored marketing approaches.

## Conclusion

Customer segmentation enables businesses to gain deeper insights into their customer base, design personalized marketing campaigns, enhance customer experience, and improve overall business performance.
